## Key Interaction Patterns

### Universal Gestures
- **Grab**: Closed fist on object
- **Throw**: Grab + quick motion + release
- **Pinch**: Thumb + index proximity
- **Swipe**: Quick directional hand movement
- **Rotate**: Two-hand opposing motion
- **Zoom**: Pinch distance change

### Voice Command Structure
- **Direct**: "Red" → Immediate action
- **Target + Action**: "Button → Click"
- **Complex**: "Show me projects from 2024"
- **Hybrid**: Voice describes, gesture executes

### Feedback Mechanisms
- **Visual**: Particle trails, glows, highlights
- **Audio**: Confirmation sounds, voice responses
- **Haptic** (future): Device vibration on grab
- **UI Toast**: Temporary command confirmation
